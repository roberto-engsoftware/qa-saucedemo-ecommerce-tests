# Projeto Q.A — Testes Manuais no SauceDemo

Este projeto tem como objetivo praticar testes manuais em um fluxo de e-commerce utilizando o site demo SauceDemo.

Foram criados cenários e casos de teste para validar funcionalidades como login, listagem de produtos, carrinho, checkout, validação de campos obrigatórios e finalização de compra.

## Site testado

https://www.saucedemo.com/

## Funcionalidades testadas

- Login
- Listagem de produtos
- Adição de produtos ao carrinho
- Remoção de produtos do carrinho
- Checkout
- Validação de campos obrigatórios
- Validação de preço total
- Finalização de compra
- Logout

## Tipos de teste realizados

- Teste funcional
- Teste negativo
- Teste de validação
- Teste exploratório

## Ferramentas utilizadas

- Google Chrome
- Windows 10/11
- GitHub
- Markdown
- Capturas de tela para evidências
